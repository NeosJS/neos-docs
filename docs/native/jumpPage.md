# 打开页面

用默认浏览器打开指定的页面。

## API

### `nativeApi.openURL(url)`
###

## 示例代码
```js
import nativeApi from '@neosjs/electron-native'

nativeApi.jumpPage('https://www.thethinkacademy.com')
```
## Params

| 参数  | 说明     | 类型   | 可选值     | 默认值 |
| ----- | -------- | ------ | ---------- | ------ |
| url | 需要打开的页面地址 | String | — | —     |
