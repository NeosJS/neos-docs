# 元素添加拖拽(窗口)

## API 接口

### `nativeApi.addElementDrag(domId)`

## 示例代码

```js
import { nativeApi } from '@neosjs/electron-ipc'

nativeApi.addElementDrag('drag')
```

## 参数

| 参数 | 说明    | 类型   | 可选值 | 默认值 |
| ---- | ------- | ------ | ------ | ------ |
| domId | 元素 ID | string | —      | —      |
