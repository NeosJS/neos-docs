# 全屏窗口 <BadgeTip text="renderer" type="green"></BadgeTip>

将当前获得焦点的窗口全屏

## API
### `nativeApi.fullScreen(flag)`
### 

## 示例代码
```js
import nativeApi from '@neosjs/electron-native'

nativeApi.fullScreen(true) // 全屏
```

## Params

| 参数  | 说明     | 类型   | 可选值     | 默认值 |
| ----- | -------- | ------ | ---------- | ------ |
| flag | 是否全屏 | Boolean | true、false | —      |
