# 加载本地文件

主窗口加载一个本地文件

## API
###  `nativeApi.loadFile(filePath)`
### 

## 示例代码
```js
import { nativeApi } from '@neosjs/electron-ipc'

nativeApi.loadUrl('/xxxx/index.js')
```

## Params

| 参数  | 说明     | 类型   | 可选值     | 默认值 |
| ----- | -------- | ------ | ---------- | ------ |
| filePath | 本地文件地址 | String | —  | —      |
