# 打开文件夹
用系统的方式打开文件夹


## API
### `nativeApi.openDirBySystem(dirPath)`
### 

## 示例代码
```js
import { nativeApi } from '@neosjs/electron-ipc'

nativeApi.openDirBySystem('/xxx/xxxx')
```

## Params

| 参数  | 说明     | 类型   | 可选值     | 默认值 |
| ----- | -------- | ------ | ---------- | ------ |
| dirPath | 文件夹路径 | String | —      | —      |
