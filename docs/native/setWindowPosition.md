# 设置窗口位置

设置当前主窗口在屏幕的x坐标和y坐标

## API
### `nativeApi.setWindowPosition(x, y)`
### 

## 示例代码
```js
import { nativeApi } from '@neosjs/electron-ipc'

nativeApi.setWindowPosition(80, 100) 
```

## Params

| 参数  | 说明     | 类型   | 可选值     | 默认值 |
| ----- | -------- | ------ | ---------- | ------ |
| x | x坐标 | Number | — | —      |
| y | y坐标 | Number | — | —      |
