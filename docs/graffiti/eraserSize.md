# 设置橡皮擦大小


## API
### `graffiti.eraserSize(size: <String>)`
### 

## 示例代码
```js
import graffiti from '@neosjs/h5-graffiti'

graffiti.eraserSize(20)
```
## 参数

| 参数                       | 说明                       | 类型   | 可选值          | 默认值       | 是否必选|
| ------------------- | -------------------- | ------ | --------------- | ------------ |------------ |
| size                | 橡皮擦大小           | Number | —               | —            | 是 |
